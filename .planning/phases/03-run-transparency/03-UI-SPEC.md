## Component Specifications

### 1. Approval Card (审批卡片)

**位置:** 嵌入聊天消息流，替代 assistant 消息中的普通内容区域

**结构:**
```
┌─────────────────────────────────────────────┐
│ 🔧 {tool_name}                              │
│ {one_line_args_summary}                      │
│                                              │
│ [允许]  [本次会话允许]  [拒绝]                │
└─────────────────────────────────────────────┘
```

**样式:**
- 容器: `Card` with `border-amber-500/40 bg-amber-500/5`
- 工具名: `text-sm font-medium text-foreground`
- 参数摘要: `text-xs text-muted-foreground truncate max-w-full`
- 按钮: `Button size="sm"` — 允许=`variant="default"`, 本次会话允许=`variant="outline"`, 拒绝=`variant="outline" className="text-destructive hover:bg-destructive/10"`
- 间距: 内部 `p-4`，元素间 `gap-3`

**状态:**
- `pending`: amber 边框，按钮可点击
- `approved`: 转为普通 Badge（绿色 "已允许"）
- `denied`: 转为普通 Badge（红色 "已拒绝"）

**过渡:** 状态变化时用 `transition-all duration-200`

### 2. Stop Button (停止按钮)

**位置:** 替换 ChatInput 中的 Send 按钮

**结构:**
- 当 `isStreaming=true`: 隐藏 Send 按钮，显示 Stop 按钮
- Stop 按钮: `Button variant="destructive" size="default"` + `Square` icon (lucide)
- 文本: `Stop`
- 点击后: 触发 abort，按钮立即切换为 disabled 状态

**间距:** 和 Send 按钮完全相同的位置和尺寸

### 3. Tool Detail Panel (工具详情展开)

**位置:** ChatMessage 内部，工具 Badge 区域下方展开

**结构 (折叠态):**
```
[tool_name 完成]  [other_tool 运行中]
共调用 2 个工具              [查看详情 ▾]
```

**结构 (展开态):**
```
[tool_name 完成]  [other_tool 运行中]
共调用 2 个工具              [收起 ▴]

┌─ tool_name ──────────────────────────────┐
│ 参数: {formatted_args}                    │
│ 返回: {truncated_result}                 │
│ 耗时: 1.2s                               │
└──────────────────────────────────────────┘

┌─ other_tool ─────────────────────────────┐
│ 参数: {formatted_args}                    │
│ 返回: (运行中...)                         │
│ 耗时: —                                   │
└──────────────────────────────────────────┘
```

**样式:**
- 展开/收起按钮: `text-xs text-muted-foreground hover:text-foreground cursor-pointer`
- 详情面板: `rounded-lg border border-border/60 bg-background/70 p-3 space-y-2`
- 参数标签: `text-xs text-muted-foreground uppercase tracking-wider`
- 参数值: `text-sm font-mono text-foreground`
- 返回值区域: `max-h-[120px] overflow-y-auto text-sm font-mono`
- 耗时: `text-xs text-muted-foreground`

### 4. Error Card (错误卡片)

**位置:** 展开的工具详情中，当 tool_error 时替代正常返回值

**结构:**
```
┌─ tool_name 错误 ─────────────────────────┐
│ ⚠ {error_message}                         │
│ {error_description}                        │
└──────────────────────────────────────────┘
```

**样式:**
- 容器: `rounded-lg border border-destructive/40 bg-destructive/5 p-3`
- 图标: `CircleAlert` from lucide, `text-destructive`
- 消息: `text-sm text-destructive font-medium`
- 描述: `text-xs text-destructive/70`

### 5. Model Selector (模型选择器)

**位置:** 左侧 sidebar，替换当前静态 Model 信息区

**结构 (关闭态):**
```
Model
[forest green dot] {current_model_name}
```

**结构 (打开态 - 第一级: Provider):**
```
Model
┌──────────────────────┐
│ ● OpenAI       ▸     │
│   Anthropic          │
│   Google             │
│   xAI                │
│   Moonshot           │
│   DeepSeek           │
└──────────────────────┘
```

**结构 (打开态 - 第二级: Model):**
```
Model
┌──────────────────────┐
│ ◂ 返回               │
│                       │
│ ● GPT 5.4  (default) │
│   GPT 4.1            │
└──────────────────────┘
```

**样式:**
- 触发器: 点击 "Model" 标题或模型名展开
- 下拉: `Card` with `p-2 space-y-1`
- 当前选中项: `bg-primary/10 text-primary rounded-md px-3 py-2`
- 未选中项: `hover:bg-secondary rounded-md px-3 py-2`
- 层级切换: 点击 provider 行进入第二级，点击 "◂ 返回" 回到第一级
- 宽度: 和 sidebar 一致 (320px)

### 6. Aborted Message (中止消息)

**位置:** ChatMessage 中，当 status='aborted' 时

**结构:**
```
{已生成的文本内容...}

─── 已中止 ───
发送新消息或重新发送
```

**样式:**
- 中止标记: `border-t border-border/60 pt-3 mt-3`
- "已中止" 文本: `text-xs font-medium text-muted-foreground uppercase tracking-wider`
- 提示文本: `text-xs text-muted-foreground`
